1.1 Android

  1. Download F-Droid from f-droid.org
  2. Open F-Droid and go to Settings, then Repositories
  3. Add a new repository by following the instructions at https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//apk/stable/repo
  4. Search for hydrapp React and panrpc Example in F-Droid and tap on Install

hydrapp React and panrpc Example should now be installed and receive updates automatically.

1.2 Debian Sid on amd64

To install the prebuilt binary package, run the following:

pkexec sudo bash - <<'EOT'
mkdir -p /usr/local/share/keyrings
curl -Lo https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//deb/debian/sid/x86_64/stable/repo.asc | gpg --dearmor --output /usr/local/share/keyrings/com.pojtinger.felicitas.hydrapp.example.react.panrpc.gpg -

cat >/etc/apt/sources.list.d/com.pojtinger.felicitas.hydrapp.example.react.panrpc.list <<EOA
deb [signed-by=/usr/local/share/keyrings/com.pojtinger.felicitas.hydrapp.example.react.panrpc.gpg] https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//deb/debian/sid/x86_64/stable debian main
deb-src [signed-by=/usr/local/share/keyrings/com.pojtinger.felicitas.hydrapp.example.react.panrpc.gpg] https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//deb/debian/sid/x86_64/stable debian main
EOA

apt update

apt install -y 'com.pojtinger.felicitas.hydrapp.example.react.panrpc'
EOT

To install the source package, build the binary package locally and install it, run the following:

pkexec sudo bash - <<'EOT'
mkdir -p /usr/local/share/keyrings
curl -Lo https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//deb/debian/sid/x86_64/stablerepo.asc | gpg --dearmor --output /usr/local/share/keyrings/com.pojtinger.felicitas.hydrapp.example.react.panrpc.gpg -

cat >/etc/apt/sources.list.d/com.pojtinger.felicitas.hydrapp.example.react.panrpc.list <<EOA
deb [signed-by=/usr/local/share/keyrings/com.pojtinger.felicitas.hydrapp.example.react.panrpc.gpg] https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//deb/debian/sid/x86_64/stable debian main
deb-src [signed-by=/usr/local/share/keyrings/com.pojtinger.felicitas.hydrapp.example.react.panrpc.gpg] https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//deb/debian/sid/x86_64/stable debian main
EOA

apt update

apt install -y dpkg-dev
apt build-dep -y 'com.pojtinger.felicitas.hydrapp.example.react.panrpc'
apt source -y --build 'com.pojtinger.felicitas.hydrapp.example.react.panrpc'
apt install -y ./com.pojtinger.felicitas.hydrapp.example.react.panrpc_*.deb
EOT

hydrapp React and panrpc Example should now be installed and receive updates automatically.

1.4 Fedora 40 on amd64

To install the prebuilt binary package, run the following:

pkexec sudo bash - <<'EOT'
dnf config-manager --add-repo 'https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//rpm/fedora/40/x86_64/stable/repodata/hydrapp.repo'
dnf install -y 'com.pojtinger.felicitas.hydrapp.example.react.panrpc'
EOT

To install the source package, build the binary package locally and install it, run the following:

pkexec sudo bash - <<'EOT'
dnf config-manager --add-repo 'https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//rpm/fedora/40/x86_64/stable/repodata/hydrapp.repo'

dnf install -y rpm-build
dnf download --source -y 'com.pojtinger.felicitas.hydrapp.example.react.panrpc'
dnf builddep -y com.pojtinger.felicitas.hydrapp.example.react.panrpc-*.rpm
rpmbuild --rebuild com.pojtinger.felicitas.hydrapp.example.react.panrpc-*.rpm
dnf install -y ~/rpmbuild/RPMS/"$(uname -m)"/com.pojtinger.felicitas.hydrapp.example.react.panrpc-*.rpm
EOT

hydrapp React and panrpc Example should now be installed and receive updates automatically.

1.6 macOS

  1. Download the .app from https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//dmg/stable/com.pojtinger.felicitas.hydrapp.example.react.panrpc.darwin.dmg
  2. Mount com.pojtinger.felicitas.hydrapp.example.react.panrpc.darwin.dmg by opening it
  3. Drag hydrapp React and panrpc Example to the Applications folder

hydrapp React and panrpc Example should now be installed and receive updates automatically.

1.7 Linux Universal (Flatpak) on amd64

To install the package, run the following:

flatpak remote-add 'com.pojtinger.felicitas.hydrapp.example.react.panrpc' --from 'https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//flatpak/x86_64/stable/hydrapp.flatpakrepo'
flatpak install -y 'com.pojtinger.felicitas.hydrapp.example.react.panrpc'

hydrapp React and panrpc Example should now be installed and receive updates automatically.

1.10 Other Platforms

hydrapp React and panrpc Example is also available as a single static binary. To install it, download the binary for your operating system and processor architecture from https://pojntfx.github.io/hydrapp/hydrapp-example-react-panrpc//binaries/stable. They include a self-update mechanism, so you should be receiving updates automatically.
